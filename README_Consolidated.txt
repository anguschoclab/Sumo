Sprint K — Consolidated Boot (created 2025-10-12 17:58)
============================================

This package includes EVERYTHING needed to start the dev server cleanly.

Install / Run
-------------
1) Unzip this into your Sumo folder:
   unzip SprintK_Consolidated.zip -d ~/Sumo

2) Start (clean optional):
   cd ~/Sumo
   ./Run.command --clean
   ./Run.command
   # or: npm run dev

Hotkeys
-------
D = advance day
W = advance week
S = (stub) save

Notes
-----
• Icons are valid tiny PNGs to quiet PWA warnings.
• Economy/persist/systems are safe stubs.
• Replace these with real implementations in later sprints.
