# Sumo — Always-Playable Baseline (Vite)

This is a clean Vite + React (TypeScript) baseline matching the static **A–K** build:
- World/time engine with +Day / +Week / Next Basho
- HeaderBar HUD with stop-mode toggle
- Simple feature flags
- Local event bus (`world:change` equivalent)

## Run
```bash
npm i
npm run dev
# open http://localhost:5173
```
